## What it does

Instead of building orders manually, you tell Maximus what you want and it places them for you. Scale into a position, set up a grid, hide your order size, DCA over time. Any order type, just ask for it.

## Scale in and scale out

Tell Maximus you want to scale into a position and it suggests how to structure it. How many orders to use, where to place them, and recommended take profit and stop loss levels based on current price action.

## Grid trading

Ask Maximus to set up a grid and it creates a set of orders designed to capitalize on price volatility within a range. Useful when you expect a market to move sideways or chop within a defined zone.

## Iceberg orders

Iceberg orders hide your true position size from the market. Only a small portion of the order is visible in the order book at any time. Ask Maximus to place one and it handles the full setup.

## DCA

Ask Maximus to DCA into any position and it spaces your entries over time or price levels according to your instructions.

## How to use it

Just tell Maximus what you want to do. Type or speak it in the prompt bar the same way you'd explain it to another trader. You don't need to know the exact order type or fill in any forms.

A few examples:

> Scale me into a long BTC position with 10% of my portfolio
> Set up a grid on ETH between 1800 and 2000
> Place an iceberg order to buy 5000 USDC of SOL
> DCA me into BTC over the next 5 days
> Long SOL with a scaled entry and set my stop loss below the last support level
> Close 50% of my ETH position at market
> Set a take profit on my BTC trade at 85,000

The conversation can go back and forth. You can ask Maximus to adjust, confirm before placing, or explain what it's about to do.

## Order type support

Every order type available on HyperLiquid is supported through Maximus. On top of that, Maximus adds order types HyperLiquid doesn't offer natively, like grid trading, scaled entries, iceberg orders, and DCA. If you can trade it on HyperLiquid, you can ask Maximus to handle it.
